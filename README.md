# Dynamic-News-Website-API-Integration-
A dynamic, responsive News Web Application built with HTML, CSS, and JavaScript. It integrates a third-party RESTful API to fetch, parse, and display real-time global news data dynamically on the frontend.

## 📝 Project Overview
A fully responsive, dynamic web application designed to fetch and display real-time news articles from across the globe. This project was built during my internship at the Digital Empowerment Network to demonstrate proficiency in asynchronous JavaScript, API integration, and dynamic DOM manipulation.

## 🚀 Key Features
* **Real-Time Data Fetching:** Integrates a third-party RESTful API to pull live news updates.
* **JSON Parsing & Rendering:** Efficiently parses JSON responses and dynamically generates HTML components to display article titles, images, and summaries.
* **Responsive UI/UX:** Clean, modern frontend built with HTML5 and CSS3, ensuring cross-browser compatibility and a seamless mobile viewing experience.
* **Asynchronous Operations:** Utilizes modern JavaScript `fetch()` / AJAX methods to ensure the page updates without requiring a full reload.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Architecture:** REST API Integration, Asynchronous Programming

## 💡 Learning Outcomes
This project heavily improved my understanding of handling endpoints, processing JSON data streams, and managing UI states—skills that are foundational for my current transition into Data Analytics and integrating Machine Learning API endpoints.
